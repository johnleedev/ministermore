# 모바일 주보 탭/결제 변경 안내

## 1) 탭 구성
- 모바일 주보 탭은 아래 4개로 구성됩니다.
  - 메인
  - 예배순서
  - 교회소식
  - 헌금자명단

## 2) 메인 화면 동작
- 메인에는 교회 메인 영역과 예배 리스트가 표시됩니다.
- 예배 리스트 항목을 클릭하면 해당 예배가 선택됩니다.
- 선택된 예배 기준으로 예배순서 탭을 보여줄 수 있는 흐름으로 구성했습니다.

## 3) 결제 금액 규칙
- 예배 리스트 개수에 따라 월 구독료(공급가)가 달라집니다.
- 기본 규칙
  - 최소 2개: 월 5,000원
  - 3개: 월 6,000원
  - 4개: 월 7,000원
  - 이후 1개 증가할 때마다 +1,000원
- 결제 시에는 부가세 10% 포함 금액으로 계산됩니다.

## 4) 결제 후 이동
- 결제가 완료되면 `BulletinCreate` 단계는 건너뜁니다.
- 즉시 `BulletinComplete` 화면으로 이동합니다.
- 완료 문구는 `마이페이지에서 관리하시면 됩니다`로 표시됩니다.
